# Manpower ERP Dashboard

A modern, clean and responsive **Manpower Management ERP Dashboard** built with React + TypeScript + Vite + shadcn/ui.

**Personal Project** — Portfolio & Learning Purpose

## ✨ Features

- Modern & Professional UI/UX
- Fully Responsive Design (Desktop + Mobile)
- Dark/Light Mode Support
- Built with TypeScript for type safety
- Fast performance with Vite
- Reusable component architecture

## 🛠 Tech Stack

- **React 18** + **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS** + **shadcn/ui**
- **React Router**
- **Lucide React** (Icons)

```bash
# 1. Clone the repo
